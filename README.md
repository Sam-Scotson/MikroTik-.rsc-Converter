## MikroTik-.rsc-Converter
Work In Progress <br>
First project as python coder and for personal use as a Network Engineer <br>
File converter for working with the difficult .rsc file format native to MikroTik RouterOS <br>
Currently allows conversion of .rsc files to .csv and .json <br>
Working on full sql .db conversion <br>
Future conversion back into .rsc planned to allow full intergration with in-house data-frames
